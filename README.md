# rnw

A general-purpose project generator written in Rust.

[![License](https://img.shields.io/crates/l/rnw.svg)](https://github.com/conao3/rust-rnw/blob/master/LICENSE)
[![Crates.io](https://img.shields.io/crates/v/rnw.svg)](https://crates.io/crates/rnw)

## Overview

rnw is a command-line tool that helps you quickly scaffold new projects with sensible defaults and best practices.

## Installation

### From crates.io

```bash
cargo install rnw
```

### From source

```bash
git clone https://github.com/conao3/rust-rnw.git
cd rust-rnw
cargo install --path .
```

## Usage

```bash
rnw
```

Follow the interactive prompts to generate your project.
